Sistema Web – Projeto Acadêmico em Equipe

Projeto desenvolvido em equipe durante o curso Técnico em Desenvolvimento de Sistemas, com o objetivo de aplicar conceitos de programação, banco de dados e desenvolvimento web.

Tecnologias Utilizadas
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

Funcionalidades
- Cadastro de informações
- Listagem de dados
- Edição e exclusão (CRUD)
- Integração com banco de dados MySQL
- Interface web simples e funcional

Trabalho em Equipe
Projeto desenvolvido em equipe.  
Minhas contribuições envolveram:
- Implementação da lógica em PHP
- Integração com banco de dados MySQL
- Desenvolvimento de funcionalidades CRUD
- Apoio no desenvolvimento do front-end

Como Executar o Projeto
1. Clonar o repositório
2. Configurar o banco de dados MySQL
3. Ajustar as credenciais de conexão
4. Executar o projeto em um servidor local (ex: XAMPP)
